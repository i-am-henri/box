## The online documentation
We provide a documentation of our service at the "docs" subdomain. The documentation is build with [Astro](https://astro.build) and [mdx](https://mdxjs.com). We use a custom build ui library, you can find the components under the folder "components" in the src-folder. [Shiki](https://shiki.style) is used for the code highlighting. Animations are implemented with the [framer-motion](https://framer.com/motion) library and [React](https://react.dev) components. Fonts are from [JetBrains](https://jetbrains.com) and [vercel](https://vercel.com), we use in particullar the JetBrains Mono font for code hightlighting and the [vercel geist font](https://vercel.com/font) for normal text.
