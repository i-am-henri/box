## Optimized for togo
Our platform is entirely responsive, made for mobile devices and tablets. This means, you can publish your code to platforms like GitHub, and continue working on your tablet later. 

## Docs
As you may already noticed, we have to folders for docs. One on the root of this project and one in the *packages* folder. Both are docs. The folder on the root are the docs for this readme file, so we can link to other files and don't have a big Readme on the root, and the other folder contains an astro project with the docs online.
